It was not only the bold simplification of the drawing which showed so rich and so singular a personality; it was not only the painting, though the flesh was painted with a passionate sensuality which had in it something miraculous; it was not only the solidity, so that you felt extraordinarily the weight of the body; there was also a spirituality, troubling and new, which led the imagination along unsuspected ways, and suggested dim empty spaces, lit only by the eternal stars, where the soul, all naked, adventured fearful to the discovery of new mysteries.
